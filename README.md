# 🏆 Sports Chaos-System Prediction Framework (SCSPF)

> **把"冷门"从异常值变成可计算的潜在因子。**
> 一套针对低比分、高偶然性、单场淘汰制体育赛事（足球/世界杯原型）的四层混沌系统预测方法论。

## 这是什么 / What is this

传统足球预测要么堆砌体测数据，要么只信博彩赔率。本框架认为：**足球的胜负手往往不在战术板，而在体能账本和场外变量**。

我们不把球员当数据机器，而把比赛当作
**「体能消耗 + 战术克制 + 场外干扰 + 裁判尺度」的混沌系统**，
通过引入疲劳衰减、对位克制、更衣室化学、转场劳损与 VAR 变量，
把冷门概率从"异常值"修正为**可计算的潜在因子**。

## 四层模型 / The Four Layers

| 层 | 名称 | 关键创新因子 |
|---|---|---|
| ① | 球员"超维度"竞技能力 | 70'后疲劳衰减率、逆足精度偏差、压迫成功率、大赛经验系数 |
| ② | 球队"生态与战术纵深" | 地理/时差适应、更衣室帮派系数、主帅战术执念画像、定位球权重 |
| ③ | 动态"战术对位与变量"模拟 | 田忌赛马对位链、5 换人 X 因素、红黄牌减员、裁判风格因子 |
| ④ | 概率化"不确定性与实时校准" | 三元结构(胜/平/负+加时+点球)、旅程疲劳扰动、场外噪音指数、VAR 修正 |
| ⑤ | **红队核验（审计层）** | 遗漏因子扫描、单点敏感度测试、伪因子排除、置信区间输出 |

> 第 ⑤ 层是让预测"经得起打脸"的关键：完成四层后，**主动寻找可能颠覆结论的因子**。

## 核心产出 / Deliverables

一个预测不是"谁赢"的断言，而是：
1. **概率分布**——常规时间胜/平/负 + 加时 + 点球的三元结构；
2. **置信区间**——而非单点值；
3. **冷门路径清单**——"若 X、Y、Z 同时发生，则冷门概率突破阈值"。

## 实战样本 / Case Studies

### 案例 1：法国 vs 英格兰（2026 世界杯四分之一决赛）

`cases/2026-world-cup-qf-france-england.md` 是框架 v2.0 的首个完整应用：
用五层模型推演 2026 美加墨世界杯四分之一决赛（法国 🇫🇷 vs 英格兰 🏴󠁧󠁢󠁥󠁮󠁧󠁿），
最终判断**法国以 54%（置信区间 48–60%）晋级**，但英格兰拥有清晰的爆冷路径
（定位球优势 + 常规时间首先得分 + 防守堡垒模式激活）。

**核心发现**：
- 这是一场 coin-flip 级别的对决，双方差距在统计误差范围边缘
- 法国凭借锦标赛 DNA、替补深度、加时赛体能管理获得微弱边际
- 英格兰定位球体系是真正的差异化武器（38-42% 进球来自定位球）
- Mbappé vs Alexander-Arnold 对位链是比赛最大变量

## 适用边界 / Scope & Limits

- 仅适用于**低比分、高偶然性、单场淘汰制**赛事；其他运动需重新标定层权重。
- "超维度"变量（如大场面先生属性）不可量化，黑天鹅（早期丢球、红牌减员、天气中断）无法建模。
- 本框架用于**分析与研究，不构成任何投注建议**。
